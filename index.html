โปรแกรมเดียวที่คลินิกต้องการ

มีฟังก์ชั่นรองรับทุกส่วนงานของคลินิก ตั้งแต่พนักงานต้อนรับ ที่เคาน์เตอร์ ไปห้องตรวจ สั่งยา ระบบสต็อก ถึงแคชเชียร์ ครบ จบในที่เดียว

ระบบนัดหมาย
  • ระบบนัดหมาย (Booking)
  • Booking Online
  • Booking by Staff
  • สร้างนัดใหม่
  • แก้ไขนัด
  • ยกเลิกนัด
  • บันทึกข้อมูลสถิติ ของลูกค้า
  • แสดงข้อมูลสถิติ ของลูกค้า ประกอบการนัด
ระบบคิวหน้าห้อง
  • ระบบคิวหน้าห้อง (Queue)
  • ฟังก์ชั่น จัดการคิวหน้าห้อง
  • สถิติการรอของลูกค้า (Waiting time)
  • สถิติเวลาการตรวจของแพทย์
ระบบบริหารจัดการคนไข้
  • ระบบบริหารจัดการคนไข้
  • รองรับ Card Reader (PC)
  • รองรับ Card Reader (iOS)
  • รองรับ Card Reader (Android)
  • Self-register
  • Check-in by QR code
  • ประวัติคนไข้ 2 ภาษา (Thai, English)
เวชระเบียน, OPD Card
  • ระบบเวชระเบียน
  • Real Folder UI
  • Private ink
  • Multi-page comparison
  • รองรับ OPD
  • รองรับ IPD
  • Print เอกสารออกได้
  • Export เป็น PDF ได้
  • Document version history
ระบบสต็อกสินค้า
  • ระบบสต็อกสินค้า
  • รองรับคลินิกหลายสาขา
  • สต็อกกลาง
  • Bulk import
  • Export by Excel
ระบบสั่งยา
  • ระบบ Order (สั่งยา, หัตถการ, Lab, อื่นๆ)
  • Order items ได้หลายประเภท
  • มี Preset ฉลากยา
  • สามารถ Tag วิธีการชำระได้
ระบบชำระเงิน ออกใบเสร็จโลโก้คลินิก
  • ระบบการคิดเงิน
  • คำนวณ Fees and Charges
  • ออกใบเสร็จ
ระบบรายงานยอดรับชำระ และรายงานต่างๆ
  • แสดงรายงานยอดรับชำระเงิน
  • Export reports in Excel
